🔧 Google Sheets Setup Guide

This page logs issues directly to a Google Sheet using a Google Apps Script. Follow these steps once to connect them.

Step 1 — Create your Google Sheet

Create a new Google Sheet and name the first row with these exact column headers (in order):

Date/Time | Resident Name | How Logged | Issue | Action Taken | Resident Response | Cost | Media | Other Comments | Last Updated | Logged By

Step 2 — Add the Apps Script

  1. In your Sheet, go to Extensions → Apps Script
  2. Delete any existing code and paste the script below
  3. Replace YOUR_SHEET_ID with your Sheet's ID (from its URL)
  4. Click Save, then Deploy → New deployment
  5. Set type to Web App, execute as Me, access Anyone
  6. Copy the Web App URL and paste it into the config bar above
function doPost(e) {
  var ss = SpreadsheetApp.openById("YOUR_SHEET_ID");
  var sheet = ss.getSheets()[0];
  var data = JSON.parse(e.postData.contents);
  sheet.appendRow([
    data["Date/Time"],
    data["Resident Name"],
    data["How Logged"],
    data["Issue"],
    data["Action Taken"],
    data["Resident Response"],
    data["Cost"],
    data["Media"],
    data["Other Comments"],
    data["Last Updated"],
    data["Logged By"]
  ]);
  return ContentService
    .createTextOutput(JSON.stringify({result:"success"}))
    .setMimeType(ContentService.MimeType.JSON);
}

function doGet() {
  return ContentService
    .createTextOutput(JSON.stringify({status:"ok"}))
    .setMimeType(ContentService.MimeType.JSON);
}

Step 3 — Media Attachments (optional)

For file uploads to work, save files to Google Drive first and paste the share link into the Media field manually, or extend the Apps Script to accept base64 files and save them to Drive. The form currently records filenames as a comma-separated list.

Step 4 — Host on GitHub

  1. Create a GitHub repo and upload index.html
  2. Go to Settings → Pages, set source to main branch
  3. Your page will be live at https://yourusername.github.io/reponame

The Apps Script URL is saved in your browser's localStorage so you won't need to re-enter it each session.
