🔧 Google Setup Guide

One-time setup to connect this page to your Google Sheet and Drive folder. Takes about 10 minutes.

Step 1 — Create a Google Drive folder for media

  1. Open Google Drive and click New → Folder
  2. Name it something like Cove Building Issues – Media
  3. Open the folder — copy the folder ID from the URL:
    drive.google.com/drive/folders/THIS_PART_IS_THE_ID
  4. Keep this ID handy — you'll paste it into the Apps Script below

Step 2 — Open Apps Script from your Google Sheet

  1. Open your Cove Resident Feedback sheet
  2. Go to Extensions → Apps Script
  3. Delete all existing code and paste the script below
  4. Replace YOUR_DRIVE_FOLDER_ID with the folder ID from Step 1
    (the Sheet ID is already filled in — your sheet is hardcoded)
const SHEET_ID  = "1OYjgiJQxFW67iJZDkypSIR9lrZTlk0h3s9N1Tc5w508";
const FOLDER_ID = "YOUR_DRIVE_FOLDER_ID";

function doPost(e) {
  var data   = JSON.parse(e.postData.contents);
  var ss     = SpreadsheetApp.openById(SHEET_ID);
  var sheet  = ss.getSheets()[0];
  var folder = DriveApp.getFolderById(FOLDER_ID);

  // Upload any attached files to Drive
  var mediaLinks = [];
  if (data.files && data.files.length > 0) {
    data.files.forEach(function(f) {
      var bytes = Utilities.base64Decode(f.data);
      var blob  = Utilities.newBlob(bytes, f.mimeType, f.name);
      var file  = folder.createFile(blob);
      file.setSharing(DriveApp.Access.ANYONE_WITH_LINK,
                      DriveApp.Permission.VIEW);
      mediaLinks.push(file.getUrl());
    });
  }

  sheet.appendRow([
    data["Date/Time"],
    data["Resident Name"],
    data["How Logged"],
    data["Issue"],
    data["Action Taken"],
    data["Response to Resident"],
    data["Cost"],
    mediaLinks.join(", "),
    data["Other Comments"],
    data["Last Updated"],
    data["Logged By"]
  ]);

  return ContentService
    .createTextOutput(JSON.stringify({result:"success"}))
    .setMimeType(ContentService.MimeType.JSON);
}

function doGet() {
  return ContentService
    .createTextOutput(JSON.stringify({status:"ok"}))
    .setMimeType(ContentService.MimeType.JSON);
}

Step 3 — Deploy as a Web App

  1. Click Save (floppy disk icon or Ctrl+S)
  2. Click Deploy → New deployment
  3. Click the ⚙️ gear icon and select Web app
  4. Set Execute as: Me
  5. Set Who has access: Anyone (not "with Google account")
  6. Click Deploy
  7. Click Authorize access → if warned, click Advanced → Go to (project name) — this is normal for personal scripts
  8. Copy the Web App URL — it ends in /exec

Step 4 — Paste the Web App URL into this page

Open index.html in a text editor, find this line near the bottom:

const SCRIPT_URL = "https://script.google.com/macros/s/AKfycbyu2JM_H4RdgPVir7GtE2qr6SrU8MbKmA36D8Gj-axUX8iwgynfs0VcXI8R0tmZUk-xiQ/exec";

Replace PASTE_YOUR_WEB_APP_URL_HERE with the URL you just copied, save the file, and re-upload it to GitHub.

⚠️ Re-deploying: If you ever edit the Apps Script, you must create a New deployment each time (not update the existing one) to get an updated URL. Old deployments keep working but won't reflect script changes.

Step 5 — Host on GitHub Pages

  1. Create a GitHub repo and upload index.html
  2. Go to Settings → Pages, set source to main branch
  3. Your page will be live at https://yourusername.github.io/reponame

How media uploads work

When you attach a file and submit the form, the file is sent to the Apps Script, saved into your Drive folder, made view-only shareable, and its Drive link is written into the Media column of your sheet — so you can click straight through from the sheet to the file.
